# MEE7
Personal discord bot with a variety of features

# Features
- Startup game servers that are located on a pterodactyl panel instance (Through DerpsCubedAPI)
- Provide information about these game servers, such as version, mods, address, etc (Through DerpsCubedAPI)
- Music playback in voice channels using Lavalink/Wavelink
- Miscellaneous fun commands

# Requirements
- Python 3.11+
- Access to a [Lavalink server](https://github.com/lavalink-devs/Lavalink). (Find a public server or host your own)
